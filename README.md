## 制作本地文档

### Html 文档

主要步骤为：

* 克隆本项目
* 创建虚拟环境（示例使用pipenv）
* 安装依赖
* 生成文档（文档生成在 _bulid 目录下)

命令示例：

```shell
git clone git@github.com:waynerv/flask-login-cn.git
cd flask-login-cn/
pipenv install
cd docs/
make html
```
